## v1 Requirements

All requirements completed and validated.

### Clipboard

- [x] **CLIP-01**: User can copy current todo text to system clipboard
- [x] **CLIP-02**: User can press `y` key in Navigate mode to trigger copy
- [x] **CLIP-03**: User sees status bar confirmation after successful copy
- [x] **CLIP-04**: User sees error message when clipboard is unavailable

### Scrolling & Mouse

- [x] **SCROLL-01**: Todo list scrolls automatically when cursor moves outside viewport
- [x] **SCROLL-02**: User can scroll using mouse wheel
- [x] **SCROLL-03**: Mouse clicks select correct item at any scroll position
- [x] **SCROLL-04**: Scroll position indicator shows current position in list title

### Priority

- [x] **PRIO-01**: User can assign priority levels (P0/P1/P2) to todos
- [x] **PRIO-02**: User can press `p` key to cycle through priority levels
- [x] **PRIO-03**: Plugin menu accessible via `P` (capital P) instead of `p`
- [x] **PRIO-04**: Priority is visually indicated in TUI with colored badge
- [x] **PRIO-05**: User can press `s` to sort todos by priority

### Plugin Configuration (added during milestone)

- [x] **CONFIG-01**: MCP server uses portable paths with ${CLAUDE_PLUGIN_ROOT}
- [x] **CONFIG-02**: Plugin metadata complete per Anthropic docs
- [x] **CONFIG-03**: README updated with installation instructions

### Self-Upgrade (added during milestone)

- [x] **UPGRADE-01**: Pressing Y in upgrade modal downloads new release binary
- [x] **UPGRADE-02**: Progress bar shows download progress
- [x] **UPGRADE-03**: Restart prompt shown after download completes
- [x] **UPGRADE-04**: Y at restart prompt replaces binary and relaunches
